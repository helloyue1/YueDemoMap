# 护理服务板块使用说明

## 尊敬的老年朋友，您好！

本说明将帮助您了解如何使用App中的"护理服务"板块，让您轻松享受贴心的护理服务。

---

## 一、护理服务板块是什么

**护理服务板块**是养老App中的一个功能模块，点击首页的"护理服务"图标即可进入。

### 🎯 这个板块能帮您做什么

| 功能 | 对您的好处 |
|------|-----------|
| **查看护理安排** | 提前知道什么时候有护理服务 |
| **了解服务内容** | 清楚知道护理人员会为您做什么 |
| **认识护理人员** | 知道是谁来为您服务，更安心 |
| **查看服务记录** | 回顾已经接受过的护理服务 |
| **评价服务质量** | 对服务进行评价，帮助我们做得更好 |
| **紧急呼叫** | 需要时快速联系护理人员 |

## 二、如何进入护理服务板块

### 操作步骤

1. **打开养老App**
   - 在手机桌面找到App图标
   - 点击打开

2. **找到护理服务入口**
   - 在App首页，找到"护理服务"图标（两个握手图标）
   - 图标下方写着"护理服务"四个字

   - 用手指点击"护理服务"图标

### 界面各部分说明

#### 1. 顶部导航栏
- **返回箭头**（←）：点击返回App首页
- **标题**：显示"护理服务"
- **消息图标**（🔔）：显示通知消息，如有新消息会有红点提示

#### 2. 搜索栏
- 可以输入护理类型（如"晨间护理"）或护理员姓名进行搜索
- 适合查找历史护理记录

#### 3. 时间筛选标签
- **今日**：显示今天的护理服务
- **本周**：显示本周的护理服务
- **本月**：显示本月的护理服务
- **全部**：显示所有的护理服务记录
- 点击不同标签，下方列表会显示对应时间的服务

#### 4. 服务记录卡片
每个卡片代表一次护理服务，包含：
- **状态标识**：显示服务状态（已完成/进行中/待开始）
- **服务类型**：如晨间护理、康复训练等
- **服务时间**：开始时间和结束时间
- **护理员信息**：为您服务的护理员姓名
- **操作按钮**：查看详情、评价等

## 四、状态颜色说明

服务卡片左上角的状态标识有不同的颜色，代表不同的含义：

| 颜色 | 状态 | 含义 | 图标 |
|------|------|------|------|
| 🟢 绿色 | 已完成 | 这项服务已经完成 | ✅ |
| 🟡 黄色/橙色 | 进行中 | 这项服务正在进行中 | ⏳ |
| 🔵 蓝色 | 待开始 | 这项服务即将开始 | 📅 |
| ⚪ 灰色 | 已取消 | 这项服务已取消 | ❌ |

**💡 小提示**：看到黄色"进行中"的卡片，说明护理人员正在为您服务或即将来为您服务。

2. 点击卡片上的"评价"按钮

**💡 小提示**：
- 您的评价对我们很重要，会帮助我们改进服务
- 如果对服务不满意，请告诉我们原因，我们会及时处理

### 5.4 呼叫护理人员

**方法一：从服务详情页呼叫**
1. 进入服务详情页
2. 点击"呼叫护理员"按钮
3. 确认拨打电话

**方法二：使用App首页的紧急呼叫**
1. 返回App首页
2. 点击"紧急呼叫"按钮
3. 选择"护理站"
4. 自动拨打护理站电话

**💡 什么时候可以呼叫**：
- 到了服务时间但护理人员还没来
- 需要临时增加护理服务
- 身体不适需要帮助
- 其他需要护理人员的情况

### 5.5 查看历史护理记录

**操作步骤**：
1. 进入护理服务板块
2. 点击顶部的"本周"或"本月"或"全部"标签
3. 列表会显示对应时间段的护理记录
4. 向上滑动可以查看更多记录

**搜索特定记录**：

## 七、使用技巧和注意事项

### 7.1 实用小技巧

**技巧1：快速找到今天的服务**
- 进入护理服务板块后，默认显示的就是"今日"标签
- 不需要切换，直接查看即可

**技巧2：了解服务进度**
- 看卡片左上角的颜色标识
- 绿色=已完成，黄色=进行中，蓝色=待开始

**技巧3：及时评价服务**
- 服务完成后尽快评价
- 时间久了可能就忘记当时的感受了

**技巧4：有问题及时反馈**
- 如果对服务不满意，可以通过评价反馈
- 也可以直接呼叫护理站反映

---

### 7.2 注意事项

**关于服务时间**
- 显示的时间是计划时间，可能会有小幅调整
- 护理人员会尽量按时到达
- 如有变动，护理人员会提前通知您

**关于服务内容**
- 服务内容是根据您的护理计划制定的
- 如果需要增加或减少服务项目，请联系护理站

**关于评价**
- 请如实评价，帮助我们改进
- 评价是匿名的，护理人员看不到是谁评价的

**关于隐私**
- 您的护理记录只有您和授权的护理人员可以查看
- 请放心使用

## 八、常见问题解答

### Q1：为什么有些服务显示"已取消"？
**A**：可能的原因：
- 您当时不需要这项服务
- 护理人员临时调整安排
- 您当时不在房间
- 其他特殊情况
如有疑问，可以咨询护理站。

### Q2：如何知道护理人员什么时候来？
**A**：
1. 在护理服务板块查看"今日"安排
2. 服务开始前15分钟，App会推送提醒（如果开启了通知）
3. 护理人员到达前通常会敲门或按门铃

### Q3：可以要求提前或推迟服务时间吗？
**A**：可以。您可以通过以下方式：
1. 提前呼叫护理站说明情况
2. 护理人员来时当面沟通
3. 请家属帮忙联系护理站

### Q4：为什么有些服务没有"评价"按钮？
**A**：可能的原因：
- 服务还未完成（状态不是"已完成"）
- 服务完成时间太久（超过7天）
- 系统故障（请联系工作人员）

### Q5：看到服务状态一直是"待开始"，但时间已经过了？
**A**：
- 可能是护理人员临时有事耽搁
- 可能是系统状态更新延迟
- 建议呼叫护理站询问情况

### Q6：如何查看护理人员的资质？
**A**：
1. 进入服务详情页
2. 点击护理员头像或姓名
3. 可以查看护理员的：
   - 姓名、照片
   - 职位（护士/护理员）
   - 工作年限
   - 专业资质

### Q7：护理服务板块打不开怎么办？
**A**：
1. 检查手机网络连接是否正常
2. 尝试关闭App后重新打开
3. 重启手机后再试
4. 如果还是不行，请联系工作人员

## 九、安全提示

### 紧急情况处理

**如果您身体不适或遇到紧急情况**：

1. **立即呼叫**
   - 点击App首页的"紧急呼叫"按钮
   - 或按床头的紧急呼叫按钮（如有）
   - 或大声呼救

2. **说明情况**
   - 告诉护理人员您哪里不舒服
   - 说明紧急程度
   - 告知您的房间号

3. **等待帮助**
   - 保持冷静
   - 在原地等待护理人员到来
   - 不要自己勉强行动
